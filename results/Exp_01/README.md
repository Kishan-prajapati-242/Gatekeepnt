
# Exp 1: Zero-Shot Llama 3.1 8B Instruct
 
Baseline. The base model with no fine-tuning, evaluated on all four test sets using the same prompt format as fine-tuned experiments.
 
## Files
 
- `exp1_zero_shot.json` — aggregate metrics with bootstrap CIs
- `exp1_per_example.json` — per-example scores (for paired comparison with other experiments)
- `exp1_preds_[sells|medlane|cochrane|plaba].json` — raw model predictions per test set
